about & contact

Hi, my name is Ivana Vasilj. I work as a freelance front-end web developer.

Although I have a broad skillset as a developer, I opted for building user interfaces, because front-end development is what I love and therefore I'm best at. I can be obsessed with semantics, and care very much about pixel-perfection and overall website performance. I write optimised HTML 5 & CSS 3, and am a big fan of SCSS, CoffeeScript, and Zepto.

I use Git for version control, and Ruby on Rails as a back-end in my pet projects.
Dark secret: I used to be a Java programmer.
